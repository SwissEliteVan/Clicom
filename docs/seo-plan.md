## Performance SEO (Core Web Vitals)

### LCP (Largest Contentful Paint) < 2.5s

**Optimisations**
- Hero image : SVG léger (< 20KB) ou fond CSS
- Pas de Google Fonts (system-ui)
- CSS/JS minifiés et defer

### CLS (Cumulative Layout Shift) < 0.1

**Optimisations**
- Width/height sur toutes images/SVG
- Pas de contenu injecté tardivement au-dessus du fold
- Réserver espace pour iframe Calendly

### FID (First Input Delay) < 100ms

**Optimisations**
- JS minimal, defer
- Pas de scripts bloquants dans <head>
